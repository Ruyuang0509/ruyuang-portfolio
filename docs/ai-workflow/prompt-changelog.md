## 人類控制點

1. 申請者決定哪些作品與研究主張可以公開。
2. 申請者確認作品、媒體、credit 與個人資料的真實性。
3. AI 建議需經 repository 檢查、實際建置與瀏覽器行為驗證。
4. 未取得的素材或證據維持缺席，不以生成內容替代。
